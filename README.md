# Web SSH

## Use locally

Simply clone the project:

```bash
git clone https://github.com/238728/WebSSH.git
cd WebSSH
```

Then run: 

```bash
npm run build:start
```

And then visit [https://localhost:3000](https://localhost:3000) in your broswer to use the app.
